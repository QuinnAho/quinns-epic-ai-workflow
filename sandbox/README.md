# Sandbox

Generated games live here.

Rules:

- Each game gets its own folder: `sandbox/<game-slug>/`
- Keep game-specific HTML, JavaScript, CSS, assets, and notes inside that folder
- Keep the original generation brief in `sandbox/<game-slug>/idea.txt`
- Keep the baseline repo commit in `sandbox/<game-slug>/baseline-ref.txt`
- Keep clarification answers in `sandbox/<game-slug>/clarifications.txt`
- Keep the combined intake source in `sandbox/<game-slug>/intake.md`
- Keep the default test harness in `sandbox/<game-slug>/tests/`
- Keep release bundles in `sandbox/<game-slug>/release/` when archiving a finished game
- Keep shared workflow files at repo root: `AGENTS.md`, `STATUS.md`, `PROJECT.md`, `specs/`, and `scripts/`
- Record the active entry file in `STATUS.md`

Typical layouts:

- `sandbox/<game-slug>/index.html`
- `sandbox/<game-slug>/game/index.html`
- `sandbox/<game-slug>/public/index.html`
- `sandbox/<game-slug>/dist/index.html`
